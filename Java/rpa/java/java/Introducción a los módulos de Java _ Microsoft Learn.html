Introducción a los módulos de Java | Java para principiantes

con Brian Benz, Kirk Pepperdine

Java para principiantes
10 abr 2022

El ingeniero de software principal de Microsoft para Java y el campeón de Java Kirk Pepperdine nos presenta los módulos de Java, una característica de Java 9 y versiones posteriores. Un módulo de Java es un conjunto de paquetes y tipos de Java específicos, incluidas clases, interfaces y mucho más, empaquetados con archivos de datos y recursos estáticos.
